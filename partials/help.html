The basics

This statistical dashboard is intended to give IFLS-member libraries a site to view and compare statistics from a number of different sources. Currently, it only has some basic functionality and reports; see the enhancements section for what's planned in the future.

It's very simply to navigate, there are only three things you can do:

  1. Select a library
  2. Select a date range
  3. Select a category

Options

Each chart except Summary charts can be viewed in a number of different formats. To select a different format:

  1. Hover over the chart
  2. Find the toolbar that appears below the chart (see image below).
  3. Select the new format.

When you hover over the chart, some options for downloading the data will also appear.

Chart descriptions

Summary Tab

Summary charts are brief overviews. If you select an individual library instead of All libraries, you'll also see checkouts, checkins and renewals broken down by terminal number.

  • Wireless data is only available for those libraries that have Fortinet access points.
  • E-materials data is only available after the month has ended. Current month data is not available. Freading data will be added shortly.
  • Website data is currently only available for libraries that host their websites through IFLS. If you do not host your website through IFLS, but want to see your data here, please contact us through the Help Desk. If you selected All libraries, the website data is for the MORE catalog.

ILS Tab

The data in this section is updated from the Sierra database nightly. We intend to add more reports to this section in the future. Data from the following item/bib fields are available: owning location, location code, format, itype, and icode2. Data from the following patron fields are available: act150 location, home library, age, and ptype.

  • Note that data by day of the week is the average for that day.
  • Large data sets (i.e. Checkouts by Act150 or Age) are summarized for the charts, but if you select the Table view or download to Excel (see Options section above), you'll get the full data set.

E-materials Tab

The data in this section is imported from Overdrive monthly. I'm working with Overdrive to see if there is an easy way for us to import more current data.

  • Note that data by day of the week is the average for that day.
  • Freegal data will hopefully be added to this tab in the next few months.

Technology Tab

  • Note that data by day of the week is the average for that day.
  • Wireless data is only available for those libraries that have Fortinet access points. It is updated nightly.
  • Computer uses comes from Pharos. It is updated nightly.
  • The bandwidth charts are available for those libraries that use Fortinet routers managed by IFLS. For some libraries, an incorrect chart is displayed. This will be fixed shortly.

Websites Tab

  • Not available unless a specific library is selected
  • Website data is currently only available for libraries that host their websites through IFLS. If you do not host your website through IFLS, but want to see your data here, please contact us through the Help Desk. If you selected All libraries, the website data is for the MORE catalog.
